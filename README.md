# ai-sprite-image-generator

Use Google Nano Banana Pro via kie.ai to generate sprite images

## Installation

```bash
bun add ai-sprite-image-generator
```

## Usage

```typescript
import { greet } from 'ai-sprite-image-generator';

console.log(greet('World')); // Hello, World!
```
